## 🐾 CodePawl

CodePawl is an open-source server-side coding-agent stack for Git, CI/CD, agent tracing, and long-running engineering workflows.

Currently building:

* **OpenPawl** - coding-agent workflows for Git and CI.
* **TracePawl** - transparent execution tracing for agent runs.
* **MemPawl** - memory and handoff layer for coding agents.
* **CachePawl** - efficient inference experiments for hybrid Mamba-Transformer systems.
